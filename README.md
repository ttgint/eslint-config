# ttgint/eslint-config

ESLint config for Node.js projects

## Installation

`npm i eslint prettier @ttgint/eslint-config --save-dev`

## Setup

Add the config to either the `package.json`:

```json
{
  "eslintConfig": {
    "extends": "@ttgint"
  }
}
```

or to the `.eslintrc`

```json
{
  "extends": "@ttgint"
}
```
